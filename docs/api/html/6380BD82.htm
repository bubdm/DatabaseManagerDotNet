Remarks
 Notes to Implementers
The default value is (?<name>^.+?)([\.]+[^.]*)(sql$) .
 Important
The name format must be a regular expression which provides a named capture name . That capture is used to extract the name of the batch from the resources manifest name. The resource manifest name is typically Folder1.Folder2.Filename.Extension , e.g. MyApp.Scripts.Cleanup.sql in an assembly named MyApp with a folder named Scripts which contains an embedded assembly resource named Cleanup.sql. Using the default value, MyApp.Scripts.Cleanup would be extracted from MyApp.Scripts.Cleanup.sql .
