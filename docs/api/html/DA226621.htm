Remarks

If commands are added to CustomCleanupBatch, CustomCleanupBatch is used for cleanup instead of the batch named by CustomCleanupBatchName or the default script.

If CustomCleanupBatch is empty (has no commands) and CustomCleanupBatchName is not null, CustomCleanupBatchName is used for cleanup instead of the default script.

The default cleanup script is (each line executed as a separate command):

SQL
Copy
1DBCC SHRINKDATABASE 0

If commands are added to CustomVersionDetectionBatch, CustomVersionDetectionBatch is used for version detection instead of the batch named by CustomVersionDetectionBatchName or the default script.

If CustomVersionDetectionBatch is empty (has no commands) and CustomVersionDetectionBatchName is not null, CustomVersionDetectionBatchName is used for version detection instead of the default script.

The default version detection script is (each line executed as a separate command):

SQL
Copy
1IF EXISTS (SELECT 1 FROM INFORMATION_SCHEMA.TABLES WHERE TABLE_TYPE='BASE TABLE' AND TABLE_NAME='__@TableName') SELECT 1 ELSE SELECT 0;
2IF (SELECT count(*) FROM [__@TableName] WHERE [__@NameColumnName] = '__@KeyName') = 0 SELECT -1 ELSE SELECT 1;
3SELECT [__@ValueColumnName] FROM [__@TableName] WHERE [__@NameColumnName] = '__@KeyName';

The following replacements will be done in the default version detection script: __@TableName with DefaultVersionDetectionTable (default: _DatabaseSettings), __@NameColumnName with DefaultVersionDetectionNameColumn (default: Name), __@ValueColumnName with DefaultVersionDetectionValueColumn (default: Value), __@KeyName with DefaultVersionDetectionKey (default: Database.Version).

Therefore, the default script requires the following table setup:

SQL
Copy
1CREATE TABLE [_DatabaseSettings]
2(
3    [Id]    INT            PRIMARY KEY IDENTITY(1,1),
4    [Name]  NVARCHAR(1024) NOT NULL,
5    [Value] NVARCHAR(MAX)  NULL
6);
7
8INSERT INTO [_DatabaseSettings] ([Name], [Value]) VALUES ('Database.Version', '0');

The default setup script (as shown above) can be obtained using GetDefaultSetupScript(DbBatchTransactionRequirement, NullableIsolationLevel).
