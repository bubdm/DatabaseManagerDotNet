Remarks
 Notes to Implementers
State, Version, InitialState, InitialVersion are updated to reflect the current state and version of the database after initialization.
 Notes to Implementers
Initialize should be callable multiple times and independent of the current state and version.
 Notes to Implementers
Details about failures should be written to logs.
 Notes to Implementers
If the database is already initialized, it will be closed first (calling Close).
