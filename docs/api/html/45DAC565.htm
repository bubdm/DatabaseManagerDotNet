Gets whether the database manager supports the restore functionality.

Syntax

Copy
bool SupportsRestore { get; }

Property Value

Type: Boolean
true if the database manager supports restore, false otherwise.
Remarks
 Notes to Implementers
SupportsRestore should be set during the construction of the database manager and then remain unchanged and independent of the current database state or version (only depending on the database manager configuration).
